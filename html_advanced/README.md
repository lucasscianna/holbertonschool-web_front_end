# HTML Advanced - Techium Project

## Description
Ce projet porte sur la création d'une structure HTML complexe et sémantique pour une agence digitale appelée **Techium**. L'objectif était de construire un site web complet et un guide de style sans utiliser de CSS, en se concentrant exclusivement sur la hiérarchie des balises, l'accessibilité et les bonnes pratiques HTML5.

## Contenu du projet
Le dépôt est organisé autour de deux axes principaux :

1.  **Le Site Web (Pages principales)** :
    * Structure multi-sections (Hero, Services, Works, Testimonials, News, Contact).
    * Navigation sémantique avec listes et ancres.
    * Utilisation de balises avancées comme `<blockquote>`, `<address>` et `<small>`.

2.  **Le Styleguide** :
    * Un catalogue de tous les éléments HTML utilisés.
    * Démonstration des titres, paragraphes, listes (ordonnées, non-ordonnées, définitions).
    * Tableaux de données structurés avec attributs `scope`.
    * Intégration multimédia (Lecteurs Vidéo et Audio natifs).

## Structure des fichiers
| Fichier | Description |
| :--- | :--- |
| `index.html` | Page d'accueil finale structurée. |
| `about.html` | Page de présentation de l'agence. |
| `latest_news.html` | Page listant les derniers articles. |
| `contact.html` | Page de contact. |
| `styleguide.html` | Guide de référence pour les composants HTML. |

## Technologies utilisées
* **HTML5** : Sémantique, multimédia, formulaires et tableaux.
* **Accessibilité** : Respect de la hiérarchie des titres et des attributs sémantiques.

## Apprentissages clés
* Différencier les types de listes (`ul`, `ol`, `dl`).
* Structurer un pied de page complexe avec des métadonnées (`address`, `hr`, `copyright`).
* Mettre en place des citations sémantiques avec `blockquote` et `cite`.
* Utiliser la balise `pre` et `mark` pour la mise en forme spécifique du texte.
